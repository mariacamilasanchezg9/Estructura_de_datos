# Taller de Logica Aplicada - Ejercicios 29 y 36 📐
 
Este repositorio contiene la solución al Taller de Logica aplicada en C#, enfocado en la solución de los ejercicios 29 y 36, Utilizando Programación Orientada a Objetos (POO), respetando un diagrama de clases específico.
 
## 🚀 Cómo clonar este proyecto
 
Para descargar este proyecto en tu computadora local, sigue estos pasos:
 
1. Abre tu terminal o Símbolo del sistema.
2. Navega hasta la carpeta donde deseas guardar el proyecto.
3. Ejecuta el siguiente comando de Git:
   ```bash
   git clone https://github.com/mariacamilasanchezg9/Estructura_de_datos.git 
   o 
   git clone git@github.com:mariacamilasanchezg9/Estructura_de_datos.git

⚙️ Requisitos y Ejecución
Requisitos previos:
 
Tener instalado Visual Studio Community (o cualquier IDE compatible con C#).
 
SDK de .NET (preferiblemente la versión 6.0 o superior).
 
Pasos para ejecutar el código:
 
Abre la carpeta clonada y haz doble clic en el archivo Taller_logica_aplicada.slnx.
 
Una vez que abra Visual Studio, presiona las teclas Ctrl + F5 (Iniciar sin depurar).
 
Se abrirá la consola mostrando los resultados exactos del cálculo de áreas y perímetros para cada figura geométrica, utilizando la configuración regional (en-US) para garantizar el formato numérico correcto.

Para mantener este repositorio limpio y seguir las buenas prácticas de desarrollo, no se han subido los archivos temporales ni los binarios compilados (como las carpetas bin/ y obj/)
